# TriviaProject
A Trivia game where the questions/answers are fetched from an open API

#How to run the project
- make sure you are in the folder that contains the pom.xml
- in the terminal run
**mvn spring-boot:run**
- this should run the project, if it doesn't:

#When downloading from github, the project might be called TriviaProject-main, with a folder structure like:
TriviaProject-main/
└── TriviaProject-main/
    ├── pom.xml
    └── src/
- If this is the case, please enter the inner folder by entering the following in the project terminal:
**cd TriviaProject-main**
then
**mvn spring-boot:run**

*in case that it does not run, please open Main and try running it from there if you are using IntelIJ
